CICLOPAG V2 - LOGIN, EMPRESA E CLIENTES
========================================

ORDEM OBRIGATORIA

1. Abra o painel do Supabase:
   https://supabase.com/dashboard/project/zqmxewoevqupcsqjimxz

2. Entre em SQL Editor e clique em New query.

3. Abra o arquivo PASSO_2_SUPABASE.sql deste pacote.

4. Copie todo o conteúdo, cole no SQL Editor e clique em Run.

5. O resultado esperado é:
   Success. No rows returned

6. Depois, execute ENVIAR_CICLOPAG_GITHUB.bat.

7. Aguarde a Cloudflare concluir o novo deploy.

8. Abra o endereço publicado e acesse /app.

9. Crie a primeira conta do CicloPag.

10. Caso a confirmação de e-mail esteja ativada no Supabase, confirme a conta
    pelo link recebido antes de entrar.

11. No primeiro acesso, informe o nome da empresa. O sistema criará:
    - empresa isolada;
    - usuário proprietário;
    - identidade visual inicial;
    - módulos padrão;
    - políticas RLS;
    - auditoria;
    - área de clientes.

SEGURANCA

- Não coloque service_role, sb_secret ou senha do banco no site.
- Use no frontend somente a chave sb_publishable configurada na Cloudflare.
- Cada empresa só acessa os próprios dados através das políticas RLS.

CONFIGURACAO CLOUDFLARE

Branch: main
Build: npm run build
Diretório de saída: dist
Diretório raiz: vazio
