CICLOPAG - ENVIO AUTOMATICO PARA O GITHUB

1. Extraia este ZIP.
2. Deixe o arquivo ENVIAR_CICLOPAG_GITHUB.bat em:
   C:\Users\wande\Downloads

3. Confirme que a pasta do projeto está em:
   C:\Users\wande\Downloads\CicloPag_Base_Inicial

4. Dê dois cliques no BAT.

O BAT irá localizar a pasta com package.json, sincronizar com o GitHub,
enviar para a branch main e a Cloudflare deverá iniciar o deploy automaticamente.
